データベースエンジンを変更する

データベースエンジンを変更するには「alter table テーブル名 engine=エンジン名」構文を使用します。 

alter table テーブル名 engine=エンジン名;

bookテーブルをInnoDBエンジンに変更する例です。 

alter table book engine=InnoDB;

authorテーブルをMyISAMエンジンに変更する例です。 

alter table author engine=MyISAM;
