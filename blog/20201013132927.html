replace関数 - 文字列を置換する

文字列を置換したい場合はreplace関数を使用します。戻り値は、置換後の文字列です。 

replace(対象の文字列,置換したい文字列,置換後の文字列)

replace関数でハイフンをアンダースコアに変える場合のSQLのサンプルです。 

update book set code = replace(code, "-", "_") where id = 1;
