インデックスを削除する

インデックスを削除するには「alter table ～ drop index」コマンドを使用します。 

alter table テーブル名 drop index インデックス名;

bookテーブルのbook_author_idのインデックスを削除するサンプルです。 

alter table book drop index book_author_id;
