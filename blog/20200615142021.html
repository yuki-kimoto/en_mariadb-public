MariaDBのバージョンを確認する

MariaDBのバージョンを確認するにはmysqlコマンドの「--version」オプションを使用します。 

mysql --version

バージョンが表示されます。 

mysql  Ver 15.1 Distrib 10.1.44-MariaDB, for debian-linux-gnu (x86_64) using readline 5.2
