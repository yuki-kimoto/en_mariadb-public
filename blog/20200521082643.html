データベースを作成する - create databaseコマンド

データベースを作成するには「create database」コマンドを使用します。 

create database データベース名;

データベースの中には、複数のテーブルを作成することができます。データベース単位でアクセス権限を与えることができます。

データベースを作成するサンプルです。 

create database kimoto_system;
