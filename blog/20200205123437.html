外部キー制約を削除する

外部キー制約を削除するには「alter table drop foreign key」コマンドを実行します。 

alter table テーブル名 drop foreign key 外部キー制約名

bookテーブルのbook_ibfk_1という外部キー制約を削除するサンプルです。 

alter table book drop foreign key book_ibfk_1
