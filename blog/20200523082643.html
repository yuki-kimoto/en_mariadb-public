ユーザーを作成する - create userコマンド

ユーザーを作成するには「create user」コマンドを使用します。 

create user ユーザー名;

「create user」コマンドでユーザーを作成するサンプルです。 

create user kimoto;

ユーザーにパスワードとデータベースとテーブルの権限を設定するには?

ユーザーにパスワードとデータベースとテーブルの権限を設定するには、grantコマンドを使用します。
